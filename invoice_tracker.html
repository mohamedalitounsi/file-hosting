Connect to an Excel Online workbook

One-time setup required
Live sync writes to your real Microsoft account, so it needs you to register a free Azure AD app once and host this file at a stable web address (OAuth logins can't redirect into a sandboxed preview).

1. Go to portal.azure.com → Azure Active Directory → App registrations → New registration.
2. Name it anything (e.g. "Spend Ledger"). Under "Supported account types" choose the option for your account type. Under Redirect URI, choose Single-page application (SPA) and enter the exact URL this page is hosted at (must be https, e.g. a GitHub Pages or Netlify URL).
3. After creating it, copy the Application (client) ID from the Overview page and paste it below.
4. Under "API permissions," add Microsoft Graph delegated permission Files.ReadWrite (and grant admin consent if prompted).
5. Create or pick an Excel workbook in your OneDrive, copy its sharing link (Share → Copy link), and paste it below too.
